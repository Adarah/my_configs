1 Fresh linux install

1.1 update pacman package databse

sudo pacman -Syu

1.2 get yay

sudo pacman -S git git clone https://aur.archlinux.org/yay.git cd yay makepkg -si

1.3 swap caps and escape

1.3.1 get the xcape package

pacman -S xcape

1.3.2 add this to ~/.xprofile

setxkbmap us -option ’caps:ctrlmodifier’ xcape -e ’CapsLock=Escape’

2 ZSH

2.1 install zsh if needed

sudo pacman -S zsh

2.2 get oh-my-zsh

2.3 get powerline 10k

3 neovim

3.1 install neovim

sudo pacman -S neovim

3.1.1 add these to .zshrc

export EDITOR=/usr/bin/nvim export VISUAL=/usr/bin/nvim

3.2 get init.vim from my git repo

git clone git@github.com:Adarah/myconfigs.git mkdir ~/.config/nvim cp myconfits/init.vim ~/.config/nvim/

4 emacs

4.1 install emacs (AUR)

yay -S emacs-git

4.2 get doom emacs

4.2.1 don’t forget to add it to the PATH in your zsh.rc file

export PATH=$PATH:~/.emacs/bin/

4.2.2 copy the hidden config files from my repos to .doom.d

4.3 if using lsp, don’t forget to run M-x lsp-install-server

4.4.1 py-isort

5.1 install pyenv

curl https://pyenv.run | zsh
